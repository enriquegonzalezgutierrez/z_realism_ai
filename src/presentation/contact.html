Contribution Protocol

As an MIT-licensed project, we encourage developers and researchers to fork the repository and propose technical improvements. Whether it's optimizing the AnimateDiff temporal loop or enhancing the CIELAB evaluation heuristics, your contributions help push the boundaries of photorealistic character synthesis.

How to Contribute:

  1. Fork the Repository: Establish a local research manifold of the source code.
  2. Apply Clean Code Patterns: Ensure your changes adhere to SOLID principles and the established Hexagonal Architecture.
  3. Submit for Peer Review: Open a Pull Request with detailed telemetry data if you are modifying generative weights or neural parameters.
