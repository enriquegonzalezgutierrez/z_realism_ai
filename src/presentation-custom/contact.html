Contribution Protocol

As an MIT-licensed project, we encourage developers to fork the repository and propose improvements. Whether it's optimizing the AnimateDiff temporal loop or enhancing the Cyber-Capsule UI, your contributions help push the boundaries of 2D-to-Realism synthesis.

How to Contribute:

  1. Fork the Repository: Create a local manifold of the source code.
  2. Apply Clean Code Patterns: Ensure your changes adhere to SOLID principles and DDD architecture.
  3. Pull Request: Submit your changes for peer review. Please include detailed telemetry data if you are modifying generative weights.
